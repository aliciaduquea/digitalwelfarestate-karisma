Esta matriz complementa el análisis de responsabilidades al mostrar, de forma sintética, las funciones que desempeña cada actor durante el funcionamiento del RUI. Mientras la matriz RACI identifica el nivel de responsabilidad de cada institución en una actividad específica, esta herramienta permite visualizar la distribución general de funciones, distinguiendo quién diseña el sistema, quién lo implementa, administra y verifica la información, realiza la clasificación, utiliza sus resultados o interactúa directamente con la ciudadanía.

Para facilitar su lectura, se marca con una casilla llena cuando una actividad forma parte de las responsabilidades de una entidad. También pueden incorporarse convenciones adicionales para representar otros tipos de participación, como brindar apoyo técnico, suministrar información o solicitar un servicio. Su uso facilita identificar concentraciones de funciones, posibles vacíos institucionales y oportunidades para fortalecer la coordinación.
